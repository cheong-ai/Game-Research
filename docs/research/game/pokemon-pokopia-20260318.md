# 포켓몬 포코피아(Pokemon Pokopia) 리서치

## 기본 정보
- **개발사**: 게임프리크(Game Freak) + 코에이테크모 오메가포스(Koei Tecmo Omega Force)
- **퍼블리셔**: 닌텐도 / The Pokemon Company
- **출시일**: 2026년 3월 5일
- **플랫폼**: Nintendo Switch 2 독점
- **장르**: 생활 시뮬레이션 / 샌드박스 크래프팅
- **가격**: $69.99 (미국) / ¥8,980 (일본) / ₩79,800~88,000 (한국)
- **메타크리틱**: 89점 (포켓몬 시리즈 역대 최고)
- **판매량**: 출시 4일 만에 220만 장 돌파

## 핵심 루프

플레이어는 **인간으로 변신한 메타몽**을 조작하며 다음 사이클을 반복한다:

1. **탐험** — 새로운 지역/바이옴을 발견하고 야생 포켓몬과 조우
2. **교감** — 포켓몬과 상호작용하여 기술을 습득 (불꽃, 물, 바위 등)
3. **수집/가공** — 습득한 기술로 자원 채집·가공 (불 기술로 벽돌 제작, 물 기술로 식물 재배 등)
4. **건설** — 마을과 서식지를 건설·꾸미기
5. **유인** — 완성된 환경에 새로운 포켓몬 종을 유인
6. **요청 완수** — 포켓몬들의 요청(퀘스트)을 수행하여 바이옴 복원 → 새 지역 해금

**세션 구조**: 현실 시간 연동 설계로, '깨고 끝내는' 플로형이 아닌 '오래 머무는' 스톡형 소비 패턴 유도. 매일 돌아올 이유를 제공하는 동물의 숲형 리텐션 구조.

**핵심 차별점**: 전투가 전혀 없음. 포켓몬 기술(불대문자, 물대포, 바위깨기 등)이 전투 수단이 아닌 **건설·가공 도구**로 재해석됨.

## 주요 시스템

### 진행(Progression) 시스템
- **요청(Request) 기반 진행**: 메이저/마이너 요청으로 분류
  - 메이저 요청: 바이옴 전체 복원 → 새 지역 해금
  - 마이너 요청: 개별 포켓몬의 소규모 과제
- **포켓몬 모집**: 서식지 환경 조성 → 특정 종 유인 → 도감 수집
- **기술 습득**: 포켓몬과의 교감으로 메타몽이 새로운 변신 능력 획득 → 상호작용 범위 확장
- **비선형 구조**: 여러 지역을 동시에 발전시킬 수 있는 자유도 제공

### 경제 시스템
- **자원 기반 경제**: 채광, 농업, 채집 등 다양한 자원 수급 경로
  - 광석 채굴 → 제련 (불 포켓몬 활용)
  - 농업 → 수확 (토지 관리가 수확량·생태계에 영향)
  - 포켓몬 드롭 아이템 → 가공 재료
- **포켓몬 노동력**: 포켓몬이 직접 재료를 가공해주는 분업 구조
- **건축 재료 체인**: 원재료 → 가공 → 건축 자재의 크래프팅 파이프라인

### 메타게임
- **도감 수집**: 서식지 조성을 통한 포켓몬 전종 수집이 장기 목표
- **마을 꾸미기**: 건축물·인테리어·환경 디자인의 창작 자유도
- **바이옴 복원**: 각 지역 생태계 완전 복원이라는 대규모 목표
- **시니 포켓몬**: 희귀 변종 수집 요소 존재

### 소셜 시스템
- **최대 4인 협동 멀티플레이** (온라인 + 로컬)
- **3가지 멀티 모드**:
  - 친구 섬 방문/초대
  - 클라우드 아일랜드 (공용 공간에서 합류)
  - 팔레트 타운 (제한 없는 자유 협동)
- **비동기 방문**: 호스트 오프라인 시에도 섬 방문 가능 (동물의 숲 대비 개선점)
- **협동 건설**: 혼자서는 불가능한 대규모 건축물(고층 빌딩 등)을 함께 건설
- **커뮤니티 이벤트**: 가든 투어, 건축 쇼케이스 등 유저 주도 이벤트

## 수익화 구조
- **과금 모델**: 풀프라이스 패키지 판매 ($69.99), 인게임 추가 과금 없음
- **주요 아이템**: 해당 없음 (인앱 구매 없는 완성형 패키지)
- **DLC**: 출시 시점 기준 미발표
- **P2W 여부**: 해당 없음 (비경쟁 게임 + 추가 과금 없음)
- **매출 성과**: 출시 4일 220만 장. 닌텐도 주가 20% 이상 상승 견인. Switch 2 본체 품절 대란의 핵심 동력.

## 플레이어 반응

### 긍정 요소
- **포켓몬 IP의 창의적 재해석**: 전투 없이 기술을 도구로 활용하는 발상 전환에 높은 평가
- **중독성**: "귀엽고 아늑하면서도 손에서 놓기 어려운" 게임플레이
- **높은 완성도**: 메타크리틱 89점, IGN Japan 만점, 포켓몬 시리즈 역대 최고점
- **방대한 콘텐츠**: 포켓몬 도감 + 비밀 요소 + 스토리가 풍성
- **멀티플레이**: 동물의 숲 대비 개선된 비동기 방문 시스템
- **애칭 '포동숲'**: 한국 커뮤니티에서 동물의 숲급 문화 현상으로 자리잡는 중

### 부정/비판 요소
- **후반부 콘텐츠 고갈**: 초반 발견의 재미가 후반으로 갈수록 사그라듦
- **반복성**: 생활 시뮬레이션 장르 특유의 반복 작업에 대한 피로감
- **기술적 문제**: 시간이 지나면 거슬리는 몇 가지 버그/성능 이슈
- **장르 한계**: 모든 취향에 맞지 않음 — 전투·경쟁 요소를 원하는 플레이어에게는 부족

### 이탈 원인
- 바이옴 복원 완료 후 반복할 콘텐츠 부족
- 실시간 연동으로 인한 "기다림" 피로
- 전투/경쟁 콘텐츠 부재로 인한 장기 목표 희석

## 기획 인사이트

### 참고할 설계 결정
1. **기존 IP 메커닉의 재해석**: 전투 기술을 건설 도구로 전환한 발상 — IP를 새 장르로 확장할 때의 모범 사례
2. **비동기 멀티플레이**: 호스트 부재 시에도 방문 가능한 설계는 소셜 참여 장벽을 크게 낮춤
3. **요청 기반 비선형 진행**: 자유도와 목표 제시의 균형을 잡은 퀘스트 구조
4. **포켓몬 = 도구 + 동반자**: 수집 동기와 실용성을 동시에 부여하는 이중 보상 설계
5. **환경 조성 → 포켓몬 유인 루프**: "만들면 온다"는 역방향 수집 메커닉이 건설 동기를 강화

### 피해야 할 설계 결정
1. **후반부 콘텐츠 절벽**: 바이옴 복원 완료 후 엔드게임 콘텐츠가 부족 → 라이브 서비스 계획 없이 스톡형 설계 채택 시 리텐션 급락 리스크
2. **실시간 연동의 양날의 검**: 매일 접속 유도에는 효과적이나, 강제 대기 시간은 이탈 원인 — 타이머 설계 시 플레이어 통제감 보장 필요
3. **장르 전환의 기대치 관리**: 전투 없는 포켓몬에 대한 일부 팬의 실망 → IP 활용 시 기존 팬의 핵심 기대를 완전히 무시하면 리스크
