# DEFRAG 리서치

## 기본 정보
- **개발사/퍼블리셔**: BlueDot Studio (인디)
- **출시일**: 2025년 11월 12일
- **플랫폼**: PC (Steam, Windows)
- **장르**: 인크리멘탈 / 탑다운 아케이드 슈터 / 서바이벌
- **가격**: $2.99
- **데모**: 있음 (Steam 데모 페이지)
- **Steam 평가**: 94% 긍정 (362건), 최근 30일 95% 긍정 (46건) — "매우 긍정적"
- **보유자 수**: 0~20,000명 (SteamSpy 추정, 2025.11 출시 기준 초기 단계)
- **동시접속 피크**: ~73명

## 핵심 루프

DEFRAG는 Nodebuster와 같은 계보의 "소프트웨어 테마 인크리멘탈 슈터"로, 다음 사이클을 반복한다:

### 코어 루프
```
섹터 진입 → 바이러스(적) 처치 → 자원 수집 → 노드 강화·조합 → 무기 모듈 장착 → 더 강한 섹터 진입
```

1. **섹터 클렌징 (액티브)** — 탑다운 시점에서 마우스로 조준, 자동 발사로 기하학적 적(바이러스) 처치
   - 웨이브 기반 진행 + 보스전
   - 적 처치 시 자원 드롭
2. **노드 강화·조합 (전략)** — 수집 자원으로 노드를 업그레이드하고, 노드끼리 조합하여 새로운 전투 스타일 생성
3. **무기 모듈 커스터마이징** — 무기에 모듈을 장착하여 성능 극대화
4. **챌린지 (보상)** — 특수 조건의 챌린지를 클리어하여 고급 보상 획득, 더 높은 난이도 해금

### Nodebuster와의 구조 비교
| 요소 | Nodebuster | DEFRAG |
|---|---|---|
| **시점** | 커서 기반 2D | 탑다운 2D |
| **조작** | 커서 이동 + 자동 공격 | 마우스 조준 + 자동 발사 |
| **성장 축** | 스킬 트리 (노드 투자) | 노드 강화 + 노드 조합 + 모듈 |
| **테마** | 시스템 바이러스 제거 | 디스크 디프래그먼트/섹터 정화 |
| **차별점** | 6종 화폐 트리 + 프레스티지 25단계 | 노드 조합 시스템 + 무기 모듈 커스텀 |

## 주요 시스템

### 진행(Progression) 시스템
- **노드 시스템**: 자원으로 노드를 강화하는 인크리멘탈 업그레이드 — Nodebuster의 스킬 트리와 유사한 위치
- **노드 조합**: DEFRAG의 핵심 차별점. 노드끼리 조합하여 새로운 전투 스타일을 만드는 크래프팅 요소
  - 단순 수치 업그레이드를 넘어 질적 변화를 유도
  - 플레이어의 빌드 다양성을 확보하는 장치
- **챌린지 시스템**: 특수 조건 미션 완수 → 보상 + 고난이도 해금
  - "Perfect Mission" 등 완벽 클리어 챌린지 존재 (스포일러 가이드가 커뮤니티에 올라올 정도의 깊이)
- **섹터 기반 진행**: 오염된 섹터를 하나씩 정화하며 새 구역 해금

### 경제 시스템
- **자원 수집**: 바이러스 처치 시 드롭되는 자원
- **노드 강화 비용**: 자원 → 노드 업그레이드에 투자
- **모듈 시스템**: 무기 성능을 세밀하게 조정하는 장착형 아이템 — 자원 투자 대상의 다양화

### 메타게임
- **빌드 실험**: 노드 조합 × 무기 모듈의 조합 공간이 리플레이 동기
- **챌린지 완주**: 전 챌린지 클리어 및 퍼펙트 미션 달성
- **실적(Achievement)**: Steam 실적 시스템 연동

### 소셜 시스템
- **없음**: 싱글플레이 전용
- Steam 커뮤니티에 퍼펙트 미션 가이드, 빌드 전략 공유 활발

## 수익화 구조
- **과금 모델**: 고정가 $2.99 일회 구매. 인앱 구매·DLC·광고 없음
- **주요 아이템**: 해당 없음
- **P2W 여부**: 해당 없음
- **매출**: 초기 단계 (출시 4개월차). 보유자 0~20K 추정 × $2.99 = 추정 총 매출 ~$60K 이하
- **데모 제공**: 무료 데모로 진입 장벽 낮춤 — 인디 인크리멘탈 장르에서 효과적인 전략

## 플레이어 반응

### 긍정 요소
- **깔끔한 인터페이스와 비주얼**: 소프트웨어 테마의 기하학적 디자인이 장르에 잘 맞음
- **노드 조합의 재미**: 단순 업그레이드를 넘어 전투 스타일 자체를 바꿀 수 있는 시스템
- **모듈 커스터마이징**: 무기 세밀 조정이 빌드 다양성과 실험 동기를 부여
- **94% 긍정률**: 출시 초기부터 안정적인 높은 평가

### 부정/비판 요소
- **정보 부족**: 출시 4개월차로 심층 리뷰·가이드가 아직 적음
- **Nodebuster 대비 차별점 인지 어려움**: 같은 "소프트웨어 테마 인크리멘탈 슈터" 계보에서 후발 주자로, 명확한 차별화 커뮤니케이션 필요
- **규모**: 보유자 수가 아직 소규모 — Nodebuster(20~50만)와 큰 격차

### 이탈 원인
- 완주형 인크리멘탈 특성상 클리어 후 자연 이탈 (장르 공통)
- 소규모 인디 타이틀의 발견성(Discoverability) 문제

## 기획 인사이트

### 참고할 설계 결정
1. **노드 조합 시스템**: Nodebuster가 스킬 트리의 "양적 성장"에 집중했다면, DEFRAG는 노드 조합으로 "질적 변화"를 추가 — 빌드 다양성 확보에 효과적인 접근
2. **무기 모듈 레이어**: 노드(캐릭터 성장) + 모듈(무기 커스텀)의 이중 커스터마이징이 의사결정 깊이를 증가
3. **챌린지 기반 난이도 분기**: 본편 진행과 별개로 챌린지 미션을 두어 하드코어 유저의 목표를 분리 — 캐주얼/코어 양쪽을 수용하는 구조
4. **데모 제공 전략**: $2.99 저가 타이틀임에도 데모를 별도로 제공하여 전환율 최적화 — "맛보기 → 구매" 퍼널 설계
5. **소프트웨어 테마의 장르 적합성**: 디프래그먼트·바이러스·섹터 등 컴퓨터 용어가 인크리멘탈의 "시스템 최적화" 판타지와 완벽히 맞물림

### 피해야 할 설계 결정
1. **후발 주자의 차별화 부족 리스크**: Nodebuster(97%, 20~50만)가 장르를 선점한 상태에서, 유사한 테마·가격·구조로는 발견성 확보가 어려움 → 노드 조합이라는 고유 메커닉이 있지만 이를 마케팅에서 더 강조했어야 함
2. **소규모 커뮤니티의 콘텐츠 부재**: 가이드·리뷰가 적어 신규 유입 시 학습 곡선 지원이 부족 → 게임 내 튜토리얼/도움말 또는 공식 가이드로 보완 필요
3. **Nodebuster의 교훈 미반영 여부**: 후반 AFK 문제, 밸런스 붕괴 등 Nodebuster가 지적받은 부분을 DEFRAG가 개선했는지 확인 필요 — 챌린지 시스템이 이를 부분적으로 해결했을 가능성 있음
