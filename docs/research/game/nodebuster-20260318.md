# Nodebuster 리서치

## 기본 정보
- **개발사**: Goblobin (인디)
- **출시일**: 2024년 8월 13일
- **플랫폼**: PC (Steam, Windows)
- **장르**: 인크리멘탈 / 서바이벌 로그라이트
- **가격**: $2.99 (약 4,000원)
- **플레이 시간**: 3~6시간 (완주)
- **메타크리틱/유저 평가**: Steam 98/100 (13,027건, 97% 긍정)
- **보유자 수**: 200,000~500,000명 (SteamSpy 추정)

## 핵심 루프

플레이어는 두 개의 화면을 오가며 다음 사이클을 반복한다:

### 이중 화면 구조
1. **브리치 세션 (액티브)** — 기하학적 도형을 조작하여 적 웨이브에서 생존
   - 커서를 움직여 자동 발사되는 총알·폭탄·체인 라이트닝으로 적 처치
   - 적 처치 → 5종 자원 드롭
   - 바이러스가 시간 경과에 따라 체력을 소모 → 생존 제한 시간 부여
   - 웨이브 보스 처치 → 특수 화폐(코어) 획득
2. **스킬 트리 (전략)** — 획득 자원을 투자하여 영구 업그레이드
   - 6종 화폐를 사용하는 대형 노드 트리
   - 무기·체력·방어·자동화 드론 등 다양한 업그레이드 경로

### 세션 사이클
```
브리치 진입 → 적 처치·자원 수집 → 사망/웨이브 클리어 → 스킬 트리 투자 → 더 강해져서 재진입
```

### 프레스티지 시스템
- 보스 처치 후 프레스티지 레벨 상승 (최대 25단계)
- 프레스티지 ↑ → 적 강화 + 자원 수급량 증가 + 새 보스 해금
- 실패 시 이전 프레스티지로 회귀하여 파밍 가능 (무한 실패 허용)

### 엔드게임
- 충분한 코어 수집 → '랩(Lab)' 탭 해금
- 1,000 넷코인으로 '갓바이러스(Godvirus)' 배포 → 게임 종료
- **명확한 엔딩이 존재하는 유한한 인크리멘탈** (무한 모드 없음)

## 주요 시스템

### 진행(Progression) 시스템
- **노드 스킬 트리**: 6종 화폐를 사용하는 비선형 트리 구조
  - 초기: +1 데미지, +4 체력 같은 소규모 수치 업그레이드
  - 중반: % 기반 보너스 (체력 비례 데미지, 킬 당 체력 회복 등) → 급격한 파워 스파이크
  - 후반: 자동화 드론, 새 무기 해금, 뱀파이리즘 등 질적 변화
- **유연한 투자**: 특정 경로 강제 없이 자유 분배 허용, 페널티 없음
- **프레스티지 25단계**: 난이도 스케일링과 보상 증가의 균형

### 경제 시스템
- **5종 드롭 자원**: 적 색상별 다른 자원 드롭
  - 일반 적 → 기본 자원(Bits)
  - 파란 적 → 노드(Node) — 고급 업그레이드용 희귀 화폐
  - 노란 적 → 프로세서(Processor) — 화폐 변환 속도 증가
- **화폐 변환**: Bits → Netcoin 변환 (감소율 적용), 프로세서로 변환 가속
- **코어**: 보스 처치 전용 화폐 → 무기·특성 해금
- **설계 특징**: 희소 화폐(노드)의 투자 우선순위가 전략적 깊이를 만듦

### 메타게임
- **올클리어 목표**: 프레스티지 25 + 갓바이러스 배포 = 게임 완료
- **스킬 트리 100% 해금**: 모든 노드 맥스가 실질적 수집 목표
- **완주형 설계**: 무한 반복이 아닌 "끝이 있는 인크리멘탈" — 4시간 내외로 만족감 있는 완결

### 소셜 시스템
- **없음**: 싱글플레이 전용. 리더보드·멀티플레이 없음
- Steam 커뮤니티 가이드·토론이 간접적 소셜 역할 (스킬 트리 풀맵 공유 등)

## 수익화 구조
- **과금 모델**: 고정가 $2.99 일회 구매. 인앱 구매·DLC·광고 없음
- **번들**: Steam 번들(Retro Incrementals, Short and Sweet Incrementals 등)에 포함되어 할인 판매
- **주요 아이템**: 해당 없음
- **P2W 여부**: 해당 없음
- **매출 추정**: 월 매출 ~$60K (2024년 12월 기준, 약 20K 구매), 총 보유자 20~50만 명 × $2.99 = 추정 총 매출 $600K~$1.5M
- **동시접속**: 피크 1,756명 (2024.10.17) → 현재 ~35명 (원샷 완주형 특성 반영)

## 플레이어 반응

### 긍정 요소
- **"올킬러 노필러"**: 3~6시간에 쓸데없는 부분 없이 알차게 응축된 경험
- **직관적 스킬 트리**: 복잡하지 않으면서도 투자 전략의 재미가 있음
- **파워 스파이크 쾌감**: 초반 2시간의 고통스러운 소규모 성장 → % 보너스 해금 후 폭발적 강해짐
- **가성비**: $2.99에 4시간의 높은 밀도 경험
- **레트로 비주얼·사운드**: CRT 스타일 + 사운드트랙이 분위기 형성

### 부정/비판 요소
- **후반 밸런스 붕괴**: 체력 업그레이드 투자 이후 난이도가 급락, 후반부가 너무 쉬워짐
- **엔드게임 허무함**: 자원이 무의미해지는 시점 이후 1.5~2시간은 AFK 방치로 진행 가능
- **재플레이 가치 없음**: 엔딩 후 무한 모드·뉴게임+ 없이 완전히 끝남
- **바이러스 메커닉**: 초반 체력 소모 바이러스가 과도한 진행 장벽으로 작용

### 이탈 원인
- 완주 후 자연 이탈 (의도된 설계) — 동시접속 98% 감소가 이를 반영
- 초반 바이러스 체력 소모에서 좌절하는 조기 이탈
- 후반 AFK 구간의 지루함으로 완주 전 이탈

## 기획 인사이트

### 참고할 설계 결정
1. **"끝이 있는 인크리멘탈" 설계 철학**: 무한 반복 대신 3~6시간의 완결형 경험 → 콘텐츠 고갈 문제를 "의도된 엔딩"으로 해결. 짧고 진한 인크리멘탈의 가능성을 입증
2. **이중 화면(액티브+전략) 구조**: 브리치(실시간 서바이벌)와 스킬 트리(전략 투자)의 분리가 "지금 당장의 긴장감"과 "장기적 계획"을 동시 충족
3. **% 보너스 파워 스파이크**: 선형 +1/+2 → 지수적 % 보너스로의 전환이 "갑자기 강해지는" 유레카 모먼트를 만듦 — 인크리멘탈 장르의 핵심 쾌감 설계
4. **희소 화폐 우선순위 전략**: 6종 화폐 중 노드(파란)의 희소성이 "무엇부터 투자할까"라는 의사결정을 강제 → 단순한 숫자 올리기에 전략층을 추가
5. **실패 허용 프레스티지**: 높은 프레스티지에서 실패해도 이전 단계로 돌아가 파밍 가능 → 좌절감 최소화

### 피해야 할 설계 결정
1. **후반부 밸런스 붕괴**: % 보너스 해금 이후 난이도 곡선이 급락 → 전반부의 긴장감이 후반에 완전히 사라짐. 파워 스파이크와 적 스케일링의 균형 필요
2. **AFK 엔드게임**: 마지막 1.5~2시간이 방치로 진행 가능한 것은 "액티브 플레이의 의미" 상실. 엔드게임에도 플레이어 개입이 필요한 메커닉 필요
3. **재플레이 부재**: 완주 후 콘텐츠 없음 — 무한 모드, 챌린지 런, 시드 기반 변형 등 선택적 리플레이 요소가 있었다면 수명 연장 가능
4. **초반 진행 장벽**: 바이러스 체력 소모가 초보자에게 과도한 벽 → 온보딩 기간의 난이도 완화 또는 적응형 튜토리얼 필요
