# 인크리멘탈 x 로그라이트 플레이어 리서치

## 조사 범위

인크리멘탈(방치형/Idle) 게임과 로그라이트(Roguelite) 장르를 즐기는 플레이어의 심리, 행동 패턴, 인구통계를 분석하고, 두 장르가 교차하는 하이브리드 영역의 타겟 플레이어 특성을 도출한다.

---

## 1. 인크리멘탈 플레이어 프로필

### 핵심 심리 동인

| 동인 | 설명 |
|---|---|
| **축적 욕구** | 숫자가 커지는 것 자체에 쾌감. 손실 걱정 없이 쌓이는 구조가 안정감 제공 |
| **즉각적 보상** | 클릭·접속 → 즉시 보상의 도파민 루프. 낮은 노력 대비 높은 빈도의 보상 |
| **오프라인 성장** | 플레이하지 않는 동안에도 진행되는 구조가 "시간 = 자산"이라는 심리 충족 |
| **최적화 탐구** | 업그레이드 경로, 프레스티지 타이밍 등 전략적 의사결정의 깊이 |
| **낮은 인지 부하** | 다른 활동(출퇴근, 업무 중)과 병행 가능한 가벼운 참여 구조 |

### 플레이 패턴

- **세션 길이**: 1~5분의 초단기 세션, 하루 10~20회 이상 분산 접속
- **세션 구조**: 접속 → 축적 자원 수확 → 업그레이드 투자 → 종료 (수확-투자 루프)
- **장기 행동**: 프레스티지(자발적 리셋) 사이클을 주 단위로 반복
- **플레이 맥락**: 이동 중, 대기 시간, 멀티태스킹 상황에서 '배경 게임'으로 소비

### 인구통계 (adjoe/Quantic Foundry 기반)

- **연령**: 25~44세 코어, 캐주얼 모바일 유저 비중 높음
- **성별**: 상대적으로 성별 균형 (모바일 방치형 기준 여성 비율 40%+)
- **플랫폼**: 모바일 중심 (70%+), 웹/PC 보조
- **게이머 유형**: 캐주얼~미드코어. Bartle 분류상 **Achiever** 우세

### 이탈 원인

1. **콘텐츠 고갈**: 더 이상 해금할 업그레이드가 없을 때 동기 소멸
2. **프레스티지 피로**: 리셋 후 동일 과정 반복의 무의미함 체감
3. **진행 벽**: 과금 없이 넘기 어려운 구간에서 좌절 → 이탈
4. **광고 피로**: 과도한 보상형 광고가 플레이 흐름을 방해
5. **의무감 전환**: 즐거움(내적 동기)이 "해야 하는 것"(외적 동기)으로 바뀔 때

## 2. 로그라이트 플레이어 프로필

### 핵심 심리 동인

| 동인 | 설명 |
|---|---|
| **숙련 욕구** | 죽음을 통해 배우고, 점점 더 잘해지는 성장 곡선의 쾌감 |
| **무작위성의 스릴** | 매 런마다 달라지는 아이템·맵·조합이 만드는 신선함과 긴장감 |
| **시너지 발견** | 아이템·스킬 조합에서 예상치 못한 강력한 시너지를 발견하는 유레카 모먼트 |
| **완결된 세션** | 10~30분 단위로 런이 끝나는 명확한 구분감 — "한 판 더"의 유혹 |
| **메타 진행** | 런 실패에도 영구 업그레이드가 쌓여 장기적 진행감 보장 |

### 플레이 패턴

- **세션 길이**: 10~30분/런, 2~5런 연속 (총 30분~2시간)
- **세션 구조**: 런 시작 → 빌드 선택 → 실시간 전투/탐험 → 사망/클리어 → 메타 투자 → 다음 런
- **장기 행동**: 캐릭터·무기 전종 해금, 고난이도 클리어, 히든 엔딩 도전
- **플레이 맥락**: 집중 플레이. PC/콘솔에서의 "몰입 세션" 중심

### 인구통계

- **연령**: 18~35세 코어
- **성별**: 남성 비율 높으나, Vampire Survivors·Hades가 여성/캐주얼 유입 확대
- **플랫폼**: PC(Steam) 중심, 콘솔·모바일로 확장 중
- **게이머 유형**: 코어~하드코어. Bartle 분류상 **Achiever + Explorer** 우세

### 이탈 원인

1. **빌드 고착**: 최적 빌드 발견 후 런의 다양성 감소
2. **난이도 절벽**: 초반 재미 후 급격한 난이도 상승에 좌절
3. **반복 피로**: 메타 진행 완료 후 런 반복의 동기 부족
4. **시간 부담**: 런 실패 시 투자한 시간이 사라지는 좌절감 (특히 긴 런)

## 3. 하이브리드(인크리멘탈 x 로그라이트) 타겟 분석

### 교차 영역의 매력

두 장르는 겉보기에 다르지만 **핵심 보상 구조가 동일**하다:

```
[런/세션 내 임시 성장] + [런/세션 간 영구 성장] = 이중 진행감
```

- 인크리멘탈의 프레스티지 = 로그라이트의 메타 진행
- 인크리멘탈의 오프라인 수확 = 로그라이트 런 실패 후에도 남는 보상
- 인크리멘탈의 숫자 쾌감 = 로그라이트의 빌드 파워 스파이크

### 하이브리드 타겟 페르소나

#### 페르소나 A: "최적화 수확자" (30대, 모바일 중심)

- **프로필**: 직장인, 게임에 긴 시간을 투자하기 어려움
- **핵심 동기**: 짧은 접속으로도 의미 있는 진행, 전략적 최적화
- **기대**: 오프라인 성장 + 짧은 액티브 런(5~10분) + 영구 메타 진행
- **민감점**: 과금 벽, 광고 과다, 의무적 플레이 강요
- **참고 타이틀**: Idle Slayer, AFK Arena, 방치형 RPG류

#### 페르소나 B: "시너지 탐험가" (20대, PC/모바일)

- **프로필**: 게임 경험이 풍부한 코어 게이머, 새로운 조합 실험을 즐김
- **핵심 동기**: 매 런의 무작위성에서 시너지 발견, 빌드 최적화
- **기대**: 다양한 빌드 경로 + 인크리멘탈 수치 폭발 + 짧은 런 사이클
- **민감점**: 빌드 다양성 부족, 최적해 수렴, 콘텐츠 고갈
- **참고 타이틀**: Vampire Survivors, Soulstone Survivors, Brotato

#### 페르소나 C: "컬렉터 성장러" (전 연령, 멀티 플랫폼)

- **프로필**: 도감 완성·전캐릭 해금 등 수집 목표에 몰입
- **핵심 동기**: 영구 해금 콘텐츠의 방대함, 완성을 향한 점진적 진행
- **기대**: 수백 개의 해금 요소 + 런마다 새로운 발견 + 수집 진행률 시각화
- **민감점**: 수집 요소가 게임플레이에 영향 없는 장식일 때 동기 저하
- **참고 타이틀**: Hades(캐릭터 관계), The Binding of Isaac(아이템 도감)

### Quantic Foundry 동기 모델 매핑

| 동기 축 | 인크리멘탈 | 로그라이트 | 하이브리드 기대치 |
|---|---|---|---|
| **성취(Achievement)** | 매우 높음 (숫자 성장) | 높음 (클리어·해금) | 최상 — 이중 진행 |
| **도전(Challenge)** | 낮음 (패배 없음) | 높음 (죽음·재시작) | 중간 — 적응적 난이도 |
| **전략(Strategy)** | 중간 (업그레이드 최적화) | 중간 (빌드 선택) | 높음 — 이중 레이어 전략 |
| **발견(Discovery)** | 낮음 (예측 가능) | 높음 (무작위 생성) | 높음 — 런마다 신선함 |
| **사회(Social)** | 낮음 | 낮음~중간 | 낮음 — 리더보드·공유 수준 |
| **몰입(Immersion)** | 낮음 | 중간 (내러티브) | 선택적 — 서사 깊이에 따라 |

## 4. 설계 시사점

### DO: 참고할 설계 원칙

1. **이중 진행 루프 설계**: 런 내 임시 파워(로그라이트) + 런 간 영구 성장(인크리멘탈)을 명확히 분리하되 상호 강화
2. **짧은 런 + 오프라인 성장**: 액티브 런은 5~15분, 비접속 시에도 자원이 축적되는 하이브리드 세션 구조
3. **빌드 다양성 확보**: 아이템·스킬 풀을 충분히 넓혀 런마다 다른 조합이 나오도록 설계 (최소 100+ 조합)
4. **시각적 수치 쾌감**: 대미지 숫자, 자원 증가 속도 등의 피드백을 과장되게 표현 — 인크리멘탈의 핵심 쾌감
5. **점진적 해금**: 수백 개의 해금 요소를 장기간에 걸쳐 배치 — 콘텐츠 고갈을 최대한 지연
6. **프레스티지 = 런 리셋의 자연스러운 통합**: 로그라이트의 사망을 인크리멘탈의 프레스티지로 재해석

### DON'T: 피해야 할 설계 함정

1. **과금 벽으로 진행 차단**: 인크리멘탈 유저는 과금 벽에 매우 민감 — 시간 투자로 돌파 가능해야 함
2. **광고 과다**: 런 중간 강제 광고는 로그라이트의 몰입을 파괴
3. **오프라인 성장만으로 모든 것 해결**: 액티브 플레이의 의미가 사라지면 "게임"이 아닌 "타이머"가 됨
4. **최적 빌드 수렴**: 메타가 1~2개 빌드로 굳어지면 런의 무작위성이 무의미해짐
5. **의무적 일일 퀘스트 남용**: "해야 할 것"이 되는 순간 내적 동기가 외적 동기로 전환 → 이탈 가속
